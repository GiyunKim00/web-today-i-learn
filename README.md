# Today I Learn 페이지 만들기
## 0. 프로젝트 소개
> `우아한테크코스 8기 공통강의 - web`의 페이지 만들기 실습입니다.
---
## 1. 기술 스택

- HTML5
- CSS3
- JavaScript
---
## 2. 요구사항
### 2-1. 기능 요구사항

### 2-1-1. 프로필 영역
- [ ] 프로필 이미지를 `<img>` 태그로 구성한다
- [ ] 닉네임(이름)을 표시한다
- [ ] 한 줄 소개를 작성한다
- [ ] 기본 정보를 `<table>`로 구성한다
- [ ] 이름을 포함한다
- [ ] MBTI를 포함한다
- [ ] 취미를 포함한다
- [ ] 좋아하는 음식을 포함한다
- [ ] 자기소개를 `<p>` 태그로 작성한다
- [ ] 소개 글 안에 `<a>` 태그를 사용해 외부 링크를 연결한다
- [ ] GitHub 또는 블로그 링크를 포함한다

### 2-1-2. 갤러리
- [ ] 갤러리 부제를 작성한다
- [ ] 9장의 이미지를 배치한다
- [ ] 이미지들을 정사각형으로 표현한다
- [ ] 하나의 주제(여행, 음식 등)를 기준으로 구성한다

### 2-1-3. 인생 작품
- [ ] 최소 2개의 작품을 소개한다
- [ ] 목록 형태`<ul>`, `<ol>`로 구성한다
- [ ] 작품 설명을 작성한다
- [ ] 작품 중 하나에 `<video>`, `<iframe>`을 사용해 영상을 삽입한다

### 2-1-4. TIL
- [ ] TIL 섹션을 구성한다
- [ ] 날짜를 포함한다
- [ ] 학습 주제를 포함한다
- [ ] 학습 내용을 포함한다

### 2-1-5. 상단 내비게이션
- [ ] `<a>` 태그를 사용해 페이지 내 이동 링크를 구성한다
- [ ] 각 섹션으로 이동 가능하도록 anchor를 설정한다

### 2-1-6. CSS
- [ ] CSS 파일을 별도로 분리한다
- [ ] `<link>` 태그로 연결한다
- [ ] 태그 선택자 사용
- [ ] 클래스 선택자 사용
- [ ] ID 선택자 사용
- [ ] CSS Grid 또는 Flexbox를 활용한다
- [ ] 9장의 이미지를 균형 있게 배치한다
- [ ] 템플릿을 기반으로 자유롭게 스타일을 적용한다
- [ ] 카드 UI / 여백 / 색상 등을 개선한다

### 2-1-7. JS
- [ ] JavaScript 파일을 별도로 분리한다
- [ ] `<script>` 태그로 연결한다
- [ ] `<form>` 태그 구성
- [ ] 날짜 `<input>` 추가
- [ ] 제목 `<input>` 추가
- [ ] 내용 `<textarea>` 추가
- [ ] 모든 입력 요소에 `<label>` 연결
- [ ] 등록 버튼 클릭 시 이벤트 발생
- [ ] 입력 데이터를 받아 새로운 TIL 생성
- [ ] 목록에 추가
- [ ] `document.querySelector()` 사용
- [ ] DOM 요소 선택 및 제어

## 2-2. 프로그래밍 요구사항

### 2-2-1. 기본 설정
- [ ] HTML 인코딩을 UTF-8로 설정한다 (`<meta charset="utf-8">`)
- [ ] viewport 설정을 추가한다
- [ ] `<title>` 태그를 설정한다
- [ ] og:title을 설정한다
- [ ] og:description을 설정한다
- [ ] og:image를 설정한다
- [ ] `<h1>` ~ `<h6>`를 사용하여 계층 구조를 구성한다
- [ ] 모든 `<img>` 태그에 alt 속성을 작성한다

### 2-2-2. 시멘틱 태그
- [ ] `<header>`를 사용한다
- [ ] `<nav>`를 사용한다
- [ ] `<main>`을 사용한다
- [ ] `<section>`을 사용한다
- [ ] `<article>`을 사용한다
- [ ] `<footer>`를 사용한다

### 2-2-3. 제한사항
- [ ] 리소스를 상대 경로로 불러온다
- [ ] images 디렉터리에는 이미지 파일만 저장한다
- [ ] css 디렉터리에는 스타일 파일만 저장한다
- [ ] root 디렉터리에는 웹 문서만 저장한다
